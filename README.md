\# Hotel Management System



A full-stack hotel management system built using:



\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js, Express

\- Database: MySQL



\## Features

\- Room management

\- Customer management

\- Booking system

\- Employee records

\- Payment handling



\## Tech Stack

\- Node.js

\- Express

\- MySQL

\- HTML / CSS / JS
